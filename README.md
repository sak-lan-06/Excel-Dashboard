# Store Sales Dashboard 📊

## Project Overview
This project is an interactive Store Sales Dashboard created to analyze sales performance, customer purchasing behavior, and platform-wise sales insights.

The dashboard provides a clear view of monthly sales trends, customer demographics, order status, top-performing states, and sales contribution from different online shopping platforms.

Dataset used : <a href="https://github.com/sak-lan-06/Excel-Dashboard/blob/main/Store%20Dashboard.xlsx"></a>

## Objective
The main objective of this project is to analyze store sales data and identify:
- Monthly sales performance
- Customer buying patterns (Men vs Women)
- Best-performing sales channels
- Top states contributing to sales
- Order status analysis
- Product category insights

## Dashboard Features

### 1. Monthly Sales Analysis
- Displays sales trends across different months.
- Helps identify high and low sales periods.
- Tracks overall business performance over time.

### 2. Customer Analysis (Men vs Women)
- Compares purchasing behavior between male and female customers.
- Shows which customer segment contributes more to sales.

### 3. Sales by Top States
- Identifies the states generating the highest revenue.
- Helps understand regional sales performance.

### 4. Sales Channel Analysis
- Analyzes sales distribution across different platforms:
  - Amazon
  - Flipkart
  - Myntra
  - Ajio
  - Meesho
  - Others

### 5. Order Status Analysis
Tracks different order stages:
- Delivered
- Cancelled
- Returned
- Refunded

### 6. Interactive Filters
The dashboard includes filters for:
- Month
- Channel
- Category

These filters allow users to explore specific business insights.

## Tools & Technologies Used
- Microsoft Excel 
- Data Cleaning
- Data Visualization
- Dashboard Design
- Business Analysis

## Key Insights
- Women customers contribute a higher percentage of sales compared to men.
- Amazon is one of the major contributors among sales platforms.
- Maharashtra, Karnataka, and Uttar Pradesh are among the top-performing states.
- Most orders are successfully delivered compared to cancelled or returned orders.

## Project Outcome
This dashboard helps businesses understand customer behavior, track sales performance, and make data-driven decisions to improve sales strategies.
